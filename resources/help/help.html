Creating a project

Use the home screen to create a new project or open an existing one.

Editing a project

In the editor, you can add child items, remove selected items, and update metadata.

Saving

Use the Project menu to save your project to a .configured file.

Git integration

If Git-managed project creation is enabled, the application can initialize a Git repository for the project.

Save your project before using Git actions such as status or commit.
