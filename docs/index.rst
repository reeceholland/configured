CONFIGURED Documentation
========================

CONFIGURED is a desktop robotics configuration studio for creating, editing,
validating, exporting, and Git-managing structured configuration projects.

The application is built for robotics and autonomous systems projects where
configuration data needs to be readable, versioned, and exported without
manually editing JSON, XML, or ad hoc parameter files.

.. image:: screenshots/configured_sample_project.png
   :alt: CONFIGURED project editor

Project Status
--------------

CONFIGURED v0.5.0 is an early release focused on project editing, validation,
Git-backed workflows, JSON/XML export, Windows packaging, and initial Linux
build support.
